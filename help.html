📖 使用教程 · 說明書

給第一次用的家長與孩子：上面是「三步就上手」，中間是每個功能在做什麼，最下面是常見問題（麥克風、聲音、費用、更新）。

🚀 三步就上手

1
打開首頁，點最上面那條橫幅。
軟體會自動安排「今天該學什麼」——先復盤到期的句子，沒有的話就接著上次沒學完的地方。孩子不用自己選，跟著橫幅走就對了。
2
一課分 9 關，從上往下做。
聽全文 → 逐句精讀 → 生詞卡 → 語法點 → 連詞成句 → 跟讀 → 聽力題 → 背句 → 造句。每做完一關會亮一格進度條，9 格全亮＝這課完成。一次做不完沒關係，下次自動接著上次那關。
3
要用嘴巴的關卡（跟讀、背句），第一次會問麥克風——點「允許」。
說完可以直接按「✅ 我說完了」，不用乾等。想以後不再每次問，看下面〈常見問題〉第一條。

💡 每天一點點比一次背很久更有效。首頁的 🔥 連續天數就是給孩子的鼓勵。

📘 每個功能在做什麼

🏠 首頁「今日學習流」

最上面那條橫幅＝軟體幫孩子排好的今天任務，點一下直達該做的關卡。

為什麼：孩子零選擇、不糾結，家長不用盯著安排。

📅 復盤（艾賓浩斯）

學過的句子會按遺忘曲線在對的時間跳出來讓你再背一次：背對了間隔就拉長（30分鐘→1天→2天→4天→7天→15天→30天→90天），連續背對 8 次就算「牢固」畢業、不再打擾；中途背錯就打回從頭再來。首頁 📅 橫幅會顯示今天到期幾句。

為什麼：背了不復習等於白背；間隔一路拉到 90 天，是為了讓學過的東西真正變成長期記憶。

🎙️ 跟讀 / 背句（語音）

對著麥克風念，軟體聽你念得對不對、打分。念完按「✅ 我說完了」立刻出結果。

為什麼：讀出來才會說；分數只是提示，重點是敢開口。

🎧 聽力題（可開盲聽）

播一句英文，選它的意思或答一個理解問題。答案由軟體嚴格把關過，不會出現「正確答案卻算你錯」的情況。想練純聽力可開「🙈 盲聽」：句子先藏起來、只靠耳朵，聽滿 3 次才給「看一眼」（看過再答對就不計分，鼓勵先用耳朵）。

為什麼：先聽懂再開口；盲聽逼孩子真的用耳朵而不是用眼睛對答案。

✍️ 造句

用當課學到的詞自己造一句，AI 幫你看通不通、給建議。

為什麼：會用才算真的學會，不只是認得。

➕ 加一課（拍照 / 貼課文）

首頁綠色橫幅 → 拍課本照片或貼一段文字 → AI 自動做成完整 9 關精讀頁。需要填一次智譜 API Key（見〈常見問題〉費用那條）。

為什麼：課本、繪本、任何你有版權的材料都能變成精讀課，不受內建課文限制。自己加的課可以隨時 🗑 刪掉。

🔊 聲音（可選雲端真人語音）

頂部「🔊 聲音」頁可以試聽、挑一個最像真人的朗讀嗓音（英語、日語各選一個）。想更自然可下載系統「增強版」語音。

覺得系統聲還是太機械？頁面頂部有 ☁️ 雲端真人語音，可選語音來源：智譜 GLM-TTS（國內可用、直接用你做課那把智譜 key、不用再申請）或 Google（母語最自然，但要海外條件）。按「▶️ 試聽英文／日文」親耳聽發音自然不自然——好聽就打開開關，帶口音就別開、保持系統聲。開了之後單句朗讀（跟讀／背句／生詞／聽力／造句／復盤）都用它；念過的句子會快取重播不花錢；沒開或失敗會自動退回系統聲。

為什麼：發音自然是語言學習的關鍵。智譜是國內就能用的雲端語音（用已有的 key），值得試聽看看夠不夠自然。

📊 學習報告（給家長）· 週報／月報／年報

怎麼進：首頁最上面那條 🔥 連續學習（寫著「看週報 →」）點一下，就進入「學習報告」頁。

頂部兩排切換鈕：英語／日語分開看、週報／月報／年報切換。可看學習量、課程進度、最常錯的地方。（每日學習量的英日分開，是從這個功能上線後才開始分別累計；課程進度和錯題本兩邊都完整。）

為什麼：家長一眼知道學得怎樣、卡在哪，還能看長期趨勢。

🔒 家長設定（複習鎖 · 要密碼）

在哪：就在上面那個「學習報告」頁，拉到最底有一塊「🔒 家長設定」。第一次用先設一個家長密碼（4 位以上，孩子改不了）。

可切兩種規則：「可以直接上新課」（預設，不強制）或 「必須複習完才能上新課」——開了之後，只要當天有到期複習沒清完，「加新課」和沒開始的新課就會鎖住（🔒），先去復盤才解鎖；已經在學一半的課不受影響。

為什麼：怕孩子只顧著開新課、不回頭複習，家長可以強制「先還舊帳再學新的」。

🎯 總評分

每課打卡頁最上面、目錄卡片課名旁邊，會顯示這課的正確率（答對比例）和完成度（做了幾關）。分數只增不減，練得越好越高。

為什麼：一眼看出哪課學紮實了、哪課還要再練。

🏔️ 登山遊戲 + 🎉 鼓勵音效

頂部「🏔️ 登山」：每答對 1 題就往上爬 20 米，從海平面一路爬泰山、富士山……到珠峰。可以選一個登山角色、上傳自己的照片當頭像、定一座想爬的山。答對題目時還會有鼓勵音效和英文誇獎（可在「🔊 聲音」頁關掉）。

為什麼：把「答對題」變成看得見的成就，孩子更有動力。

⬅️➡️ 上一課／下一課

每課打卡頁底部有「上一課／下一課」按鈕，不用回目錄就能直接接著學下一課。

🎌 日語版

頂部可切換到日語精讀（假名標音、日語朗讀）。用法一樣，報告和登山都和英語分開統計。

🙋 常見問題

每次打開都要重新允許麥克風，能不能一勞永逸？
每次重開網頁瀏覽器都會再問一次，這是瀏覽器的安全規定，不是壞掉。想設成基本不再問，照你的設備做一次：

📱 iPhone / iPad：設定 → 通用 → 鍵盤 → 打開「啟用聽寫」；再 設定 → Safari → 麥克風 → 改「允許」。強烈建議把本站「加入主畫面」（Safari 分享鈕 → 加入主畫面），從主畫面圖示打開，權限記得最牢。
💻 Mac Safari：Safari 選單 → 設定 → 網站 → 麥克風 → 把本站設為「允許」。
💻 Mac Chrome：點網址列左邊的 🔒 → 麥克風 → 「允許」，設一次基本不再問。
朗讀的聲音很機械 / 不像真人，怎麼辦？
兩步：
① 換嗓音：頂部「🔊 聲音」頁，按 ▶️ 試聽，挑一個順耳的按「選這個」。
② 下載增強版語音（免費、更自然）：
  iPhone / iPad：設定 → 輔助功能 → 朗讀內容 → 聲音 → 英文（或日文）→ 找標「增強版 / Enhanced」的點雲朵下載。
  Mac：系統設定 → 輔助功能 → 朗讀內容 → 系統語音 → 管理語音，勾選增強版下載。
下載後回「🔊 聲音」頁，新聲音會標「較自然」，選它即可。
※ 若下載後仍沒出現，代表這台設備瀏覽器暫時只開放系統聲音，屬正常限制。
加一課要花錢嗎？怎麼準備 API Key？
內建課文免費；自己「加一課」用 AI 生成才需要智譜（bigmodel.cn）的 API Key。
準備：到 bigmodel.cn 註冊 → 控制台領 API Key → 確認帳戶已充值。Key 只存在你這台設備，不會外傳。
費用：文字用 glm-4-plus（付費，出題／講解質量好），一課大約幾分錢人民幣；拍照識字用免費視覺模型。想省錢可在「進階」把文字模型改成 glm-4-flash（免費，但聽力題會多走保底題）。
更新了但頁面沒變 / 拿不到新版怎麼辦？
正常情況普通刷新就會拿到最新版（已內建網絡優先機制）。若還是舊的，按一次硬刷新：
Mac：Cmd + Shift + R。iPhone / iPad：關掉 Safari 分頁重開，或設定 → Safari → 清除歷史紀錄與網站資料。
硬刷新一次之後，之後普通刷新即可。
自己加的課想刪掉？
首頁「我的課文」區，每課右邊有 🗑 刪除鈕。刪除會一併清掉該課的學習進度和錯題，內建課文不受影響。
換一台設備，學習紀錄還在嗎？
紀錄預設存在當前這台設備上。開啟雲端備份（首頁同步卡）後，換設備登入同一份就能接著學、週報也同步。沒開雲端就只在本機。
怎麼開啟雲端備份？（國內推薦 Gitee）
首頁「☁️ 雲端備份未開啟」卡 → 點「開啟」，選雲端服務：
· Gitee 碼雲（國內·不翻牆·推薦）：到 gitee.com 免費註冊 → 建一個私有倉庫叫 jingdu-data → 設置 →「私人令牌」勾「projects」生成令牌 → 把令牌貼進軟件。國內連線穩定。
· GitHub：同樣建私有倉庫 jingdu-data + fine-grained token，但國內連 GitHub 不穩、偶爾要重試。
令牌只存這台設備、不外傳。每台設備用同一個暱稱＝同一份進度，換機也接得上。
沒網路能用嗎？
已學過、打開過的課文可以離線複習（跟讀評分、聽力播放靠系統，通常也能用）。但「加一課」要連 AI、雲端備份要連網，這兩項需要網路。
聽力題答案還是覺得不對？
如果是硬刷新前生成的舊課，可能還帶著舊版毛病。請先按上面〈更新〉硬刷新一次，把那課刪掉重新生成一門，新版的把關機制會保證答案正確。
