不同聲音「像不像真人」差很多。按 ▶️ 試聽，挑一個你最喜歡的按「選這個」。英語、日語各選一個，全站朗讀都會用它。

☁️ 雲端真人語音 Google 神經語音，母語發音、語調自然，最像真人（每月免費額度很大，一個孩子基本用不完）。開了之後全站朗讀都用它，萬一沒聲/失敗會自動退回系統聲。
👉 怎麼申請 Google Key？（一次搞定，還能鎖死防盜用）
  1. 到 console.cloud.google.com 登入 Google 帳號（要綁一張信用卡才有免費額度；用不到就不扣款）
  2. 上方建一個「專案」→ 搜尋並「啟用」Cloud Text-to-Speech API
  3. 左側「憑證」→ 建立憑證 → API 金鑰 → 複製
  4. 防盜用（重要）：點該金鑰 →「應用程式限制」選 HTTP 參照網址，加 iamamilycc.github.io/*；「API 限制」只勾 Cloud Text-to-Speech API → 儲存。這樣別人拿到也用不了
  5. 把金鑰貼到上面按「存」，選好聲音按 ▶️ 試聽，好聽就把開關打開

費用：神經語音每月前 100 萬字免費，加上本站會把念過的句子快取起來、重播不再花錢——一個孩子日常用基本 $0。
