➕ 新增課文

拍照或粘貼一段課文，AI 自動做成精讀頁（聽全文／逐句講解／生詞／語法／連詞成句／跟讀／聽力題／背句／造句）。內容由你提供，你自己拍的、你有版權的才用。

還沒有？去 bigmodel.cn 註冊 → 控制台領 API Key，並確認帳戶已充值。key 通用。
拍照走看圖識字（免費），生成文字理解和出題走上面選的模型——不用管模型叫什麼名字，選好直接按最下面「生成」即可。
分享給家人朋友：生成「一鍵配置鏈接」（對方不用申請 key）
對方點開鏈接一次，AI 就自動配置好，之後正常使用。費用走 key 所屬的智譜帳戶（一課約幾分錢）；想停掉就到智譜控制台刪那把 key。
💡 圖片走「看圖識字」，文字/Word 走文字理解。手機鍵盤上的麥克風也能直接口述。
📚 一次要加好幾課？填完一段按「加入清單」，清空後繼續填下一課，最後一次性「生成全部」。
